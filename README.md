# TaskFlow HQ

TaskFlow HQ is a simple web-based task management dashboard built with Flask and PostgreSQL (Supabase). It allows employees to log in securely, view their assigned tasks, and update their task progress in real time.

## Core Features

- Secure user login with encrypted password verification.
- A clean, responsive dashboard showing tasks assigned to the logged-in user.
- Real-time status updates (Pending, In Progress, Completed) that save instantly to the cloud database without reloading the page.
- Secure layout that keeps database credentials hidden using environment variables.

## Tech Stack

- Backend: Python, Flask
- Database: PostgreSQL (Supabase)
- Frontend: HTML, CSS, JavaScript
